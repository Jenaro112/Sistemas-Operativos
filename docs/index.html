Trabajo Práctico - Threads, concurrencia y sincronización

Esta documentación reúne los ejercicios realizados para comprender el funcionamiento de threads, concurrencia y mecanismos de sincronización en Python, Java y C#.

A lo largo de los distintos casos se analizan situaciones reales donde múltiples hilos trabajan sobre recursos compartidos, mostrando tanto los beneficios de la programación concurrente como los problemas que pueden aparecer cuando no existe una correcta coordinación.

Casos introductorios

  • Tareas simultáneas
  • Descarga + interfaz simulada

Problemas clásicos de concurrencia

  • Race Condition
  • Deadlock
  • Starvation
  • Livelock
