Tareas Simultáneas

Las tareas simultáneas permiten que varias operaciones se ejecuten al mismo tiempo, mejorando el rendimiento y aprovechando mejor los recursos del sistema.

Orden de Ejecución

El orden de ejecución no es siempre el mismo, este puede variar cada vez que el programa se ejecuta.

¿Quién decide que hilo se ejecuta primero?

Quien decide el orden de ejecución de los hilos es el "scheduler" El sistema operativo asigna tiempo de CPU a cada hilo según su algoritmo de planificación, que puede ser basado en prioridades, round-robin, entre otros.

¿Qué sucede si se eliminan las pausas?

Si eliminamos las pausas, un hilo va a tender a ejecutarse completamente antes de que el otro hilo tenga la oportunidad de ejecutarse, porque el cambio de contexto ocurre menos frecuentemente. Se pierde la intercalación visible entre hilos.

Ejercicio

Crear 2 tareas que se ejecuten al mismo tiempo, implementando un programa que:

  • Un hilo imprima números del 1 al 10.
  • Otro hilo imprima letras de la A a la J.
  • Ambos deben ejecutarse de forma concurrente.
  • Agregrar una pequeña pausa entre impresiones para observar la intercalación.
